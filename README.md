# Collaborative Code Editor Backend

Backend for a realtime collaborative code editor.

## Tech Stack

- Bun
- TypeScript
- hono
- PostgreSQL
- Docker
- Drizzle ORM
- Better Auth
- Socket.IO

## Features

- Authentication
- Projects
- File Management
- Realtime Collaboration
- Cursor Presence
- Project Invitations

## Run

```bash
docker compose up -d
bun install
bun run dev
```
